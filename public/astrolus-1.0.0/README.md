# Astrolus - a page template built with astro and tailwindcss using tailus blocks.
#### Preview

 - [Demo](https://themewagon.github.io/astrolus/)

#### Download
 - [Download from ThemeWagon](https://themewagon.com/themes/astrolus/)

## Getting Started

1. Clone Repository
```
git clone https://github.com/themewagon/astrolus.git
```
2. Install Dependencies
```
npm i
```
3. Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
